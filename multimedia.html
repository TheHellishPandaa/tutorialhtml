Imagenes en HTML



				
Para poner una imagen simple en una página web, utilizamos el elemento <img>
				


				
Se trata de un elemento vacío (lo que significa que no contiene texto o etiqueta de cierre). El
					atributo src es obligatorio y contiene la ruta que apunta a la imagen que quieres poner en la
					página, que puede ser una URL relativa o absoluta, de la misma forma que el elemento <a>
					contiene los valores del atributo href.

				


				Por ejemplo si nuestra imagen se llama dinosaur.jpg, y está en la ubicación habitual donde
				mos el siguiente código para mostrar la imagen
				


				alojamos las imágnes: (resources/img), escrbimos el siguiente codigo para adjuntar una imagen:

				
			
<img src="resources/img/dinosaur.jpg" </img>

Texto alternativo

				

				El próximo atributo que veremos es alt. Su valor debe ser una descripción textual de la imagen
				para usarla en situaciones en que la imagen no puede ser vista/mostrada o tarde demasiado en
				mostrarse por una conexión lenta a internet. Por ejemplo, nuestro código anterior podría
				modificarse así
				
	
		<img src="resources/img/dinosaur.jpg"
		alt="La cabeza y el torso de un esqueleto de dinosaurio;
		tiene una cabeza grande con dientes largos y afilados”/>


				

				Para probar el etecto de la etiqueta alt, escribe mal el nombre del archivo al crear la etiqueta y
				observa que pasa al visualizar la página.

				La cabeza y el torso de un esqueleto de dinosauno; tiene una cabera grande con dientes largos y
				atilados

				Usamos un texto alternativo por las siguientes razones
				

				
    
					
  • El usuario tiene alguna discapacidad visual y utiliza un lector de pantalla para leer el
						contenido de la web. De hecho, disponer de texto alternativo para describir las imágenes es
						útil para la mayoría de los usuarios
    • 

					
  • Como ya hemos dicho anteriormente, es posible que se haya es
						archivo o su ruta.
    • 

					
  • El navegador no admite el tipo de imagen. Algunas personas aún usan navegadores de solo
						texto, como Lynx, que muestran el texto del atributo alt.
    • 

					
  • Quieres que los motores de búsqueda puedan utilizar este texto. Por ejemplo, los motores de
						búsqueda pueden hacer coincidir el texto alternativo con la consulta de búsqueda.
    • 

					
  • Los usuarios han desactivado las imágenes para reducir el volumen de transferencia de datos
						y de distracciones. Esto sucede especialmente en teléfonos móviles y en países en que el
						ancho de banda es limitado y caro.
    • 
				

				
Anchura y altura

				

				Puedes usar los atributos ancho (width) y alto (height) para especificar la anchura y altura de tu
				imager. Puedes encontrar el ancho y el alto de Lu imagen de diversas maneras, En windows puedes
				hacerlo examinando las propiedades del archivo de imayen, pesLaña detalles, La imayen original
				del dinosaurio tiene 400 pixeles de ancho por 341 de alto.
				Sí la imagen no se ajusta a la zona en la que la queremos
				alto mediante los atributos width y height.

				Si modificas cl tamaño haciendo uso de dichos atributos, debes saber que la imagen puede verse
				desproporcionada si no mantienes la proporción de aspecto entre el ancho y cl alto. También sí la
				escalas mucho puede verse granulada.

				La mejor manera de mantener la proporcionalidad es usar solo uno de los atributos. La imagen se
				ajusta de forma automática manteniendo la proporción. Por ejemplo vamos a mostrar la imagen
				del dinosaurio con un ancho de 250 pixeles, usando el atributo widih. Veremos como la imagen sc

Comentar imágenes con figure y figcaption

				

				HTML5 incluye dos nuevas etiquetas <fiqure> y <figcaption> que permiten vincular una figura con
				su pie de imagen, El elemento <fiqure> actua de contenedor semántico
				para la imagen y el
				elemento <figcaption> dice al navegador, o a alguna tecnología de
				apoyo, que el texto que
				contiene describe la imagen que está contenida en el elemento. Ejemplo de uso:
				
	
<figure>
<img src="resources/img/dinasaur.jpg"
alt="La cabeza y el torzo de un esqueleto de dinosaurio;
tiene una cabeza grande con dientes largos y afilados
width="250"
title="Expocicion de un T-Rex en el museo de la Universidad de Manchester"
<figcaption>
Exposición de un T-Rex en el musco de la Universidad de Manchester.
</figcaption>
</figure>




				
				
Imágenes de fondo CSS


				También puedes usar CSS para insertar imágenes en páginas web . La propiedad CSS basckground-image
				y las demás propiedades background-" se usan para controlar la colocación de la imagen de
				fondo. Por ejemplo, gara poner la imagen del dinosaurio como imagen de fondo de los párrafos
				que queramos, usando una clase CSS para ello. Fodemos definir la clase CSS fondodinosaurio asi:
				
	
.fondodinosaurio{
background-inage: url(../img/dinosaur. jpg);
}




				Y depues crear un párrafo con el atributo class igual a “fondodinosaurio”
				
	
<p class="fondodinosaurio">
</p>

       

				
Si escribes algo de texto en el parrafo observaras que la imagen de fondo
					comienza a aparezer. Sigues escribiendo y la imagen de fondo se sigue mostrando para ocupar todo el
					parrafo. Lorem ipsum dolor sit amet consectetur adipisicing elit. Quaerat, asperiores debitis
					impedit ex dolore fuga fugit iure hic tempora eligendi sit temporibus, blanditiis perferendis
					corporis deleniti minus eum, eaque aspernatur.
				

				
Si solo quisieras mostrar la imagen, porque esta no esta no aporta contenido, semantico al texto y
					solo es decorativa
					el contenedor de la misma debe tener el suficiente espacio para contenerla. Puedes emplear algunas
					propiedades adicionales CSS paralograr que la imagen se muestre proporcionada dentro de la altura
					que decidas que debe tener. A continuacion se muestra la regla CSS necesaria para conseguir mostrar

Video y Audio

					

					
Audio

					Puedes incrustar video fácilmente en tu web usando el elemento HTML5 <video>. Crea una carpeta
					videos en tu carpeta resources, copia los videos que se adjuntan y añade el siguiente código a tu
					página
					
						
						
	<video sre="resources/videos/videolocal.mp4" controls
	width="320" height="240"></video>
						
					

						
Los elementos a destacar son:

						
src: Contiene una ruta al video que deseas incrustar. Funciona de la misma manera. Los atributos
						width y height definen el ancho y alto del marco en el que se mostrará el video.

						

						controls: Los usuarios deben ser capaces de controlar la reproducción de video y audio (esto es
						especialmente crítico en personas que padecen epilepsia). Se debe utilizar el atributo controls para
						incluir la interfaz de control del browser, o construir la nuestra utilizando la JavaScript API
						apropiada. Como mínimo la interfaz debe incluir una manera de empezar y terminar la
						reproducción, y ajustar el volumen.

Audio

	<audio controls>
	<source src="resources/audio/example.mp3" type="audio/mp3">
	<source src="vipper.ogg" type="audio/ogg">
	<p>Your browser doesn't support HTML5 audio. Here is a <a href="vipper.mp3">link>
