Creación de hipervínculos

¿Qué es un vinculo?


                
 Los hipervínculos nos permiten vincular documentos a otros documentos o recursos, vincular a
                    partes específicas de documentos o hacer que las aplicaciones estén disponibles en una dirección
                    web. Prácticamente cualquier contenido web se puede convertir en un enlace que, al pulsarlo
                    (activarlo), dirija el navegador a la dirección web a la que apunta el enlace (URL).

                


                Una URL (Localizador uniforme de de recursos) puede apuntar a archivos HTML, archivos de texto,
                imágenes, documentos de texto, archivos de audio o vídeo, y cualquier otra cosa que se pueda
                mostrar en la web. Si el navegador no sabe cómo manejar el archivo, te preguntará si lo quieres
                abrir (en cuyo caso la tarea de abrirlo y manejarlo se transferirá a la aplicación nativa instalada en
                el dispositivo) o si lo quieres descargar (en cuyo caso podrás ocuparte de él más tarde).
                

                Anatomia de un enlace
                

                Un enlace básico se crea incluyendo el texto o cualquier otro contenido que queramos convertir en
                un enlace dentro de un elemento ancla<a>, proporcionando a través
                del atributo href la dirección
                web hacia dónde queremos que apunte el enlace. La url que permite crear un hipervínculo al logo
                del sitio web de nuestro instituto es:
                

                http://www.ies-azarquiel.es/images/logosazarquiel/logoiesfseif.png
                


                El atributo title permite añadir información adicional sobre dicho enlace, como por ejemplo el tipo
                de información que contiene la página. Podemos usar el atributo title para completar el
                hipervínculo anterior de forma que muestre información contextual relacionada con la imagen.
                


                El atributo target nos permite decidir si el hipervínculo se abrirá en una pestaña nueva
                (target="_blank)
                


                El código que permite crear un hipervínculo a la imagen corporativa de nuestro instituto, cuya url
                hemos mencionado antes sería:
                

                
                        
        <a href="http://www.ies-azarquiel.es/images/logosazarquiel/logoiesfseif.png"
        title="Imagen corporativa del I.E.S Azarquiel" target="_blank">Logo del I.E.S Azarquiel</a>

Urls y rutas

                


                Una localizadora uniforme de recursos (URL, de las iniciales en inglés de «Uniform Resource
                Locator») es simplemente una secuencia de caracteres de texto que definen donde está situado
                algo en la web. Por ejemplo, la página de Mozilla está ubicada en https:/ /www.mozilla.org/es-ES/.

                Las URLs utilizan rutas para encontrar los archivos. Las rutas especifican dónde se encuentra el
                archivo que buscas dentro del sistema de archivos.

URL absolutas y relativas

                

                Dos términos que encontrarás en la Web son URL absoluta y URL relativa:
                

                
URL Absoluta: Hace referencia a una dirección definida por su ubicación absoluta en la web, esta
                incluye el protocolo, el nombre del dominio y la cadena de consulta

                
//host:port/path?queryStringtfragment"

                


                
La queryString (cadena de consulta) permite enviar información adicional al servidor de forma que
                este construya la página con un contenido que depende de la información que es enviada a través
                de esa cadena de consulta.

                

                    

                
Las páginas web construidas de esta forma se denominan páginas web dinámicas puesto que su
                contenido cambia dependiendo de los parámetros de consulta. La cadena de consulta está formada
                por pares de la forma variable1=valor1&variable2=valor26......Evariablen=valorn

                    

                
Ejemplo:

                https://delphos.jccm.es/delphos/controlador?operacion=consultaralumnoá4nie=12345
                
                
Esta URL usa la cadena de consulta para informar al servidor de que la acción que quiere llevar a
                cabo es: consultar alumno y que el alumno a consultar es aquel cuyo nie es 12345

                
Las páginas de inicio de un sitio web (tipicamente index.html, index.php, index.jsp) a veces
                pueden omitirse puesto qe los servidores por defecto buscan en el directorio raiz un archivo con
                uno de esos nombres para servirlo. El archivo de inicio se puede establecer mediane políticas de
                uso del servidor web que usemos.


                
Puedes comprobar esto mediante el navegador embebido que carga visual studio code. Una vez e
                con el sitio que estamos realizando, si escribes en la barra de direcciones: http://127.0.0.1:5500/
                verlas que la página que se muestra es la index.html.
