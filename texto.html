Fundamentos de texto en HTML


                
Encabezados y Parrafos

                

                
En HTML, cada parrafo tiene que estar determinado por un <p>


               
 Ejemplo:

	
		
                
<Soy un parrafo>

	

		


                    
                
Establecemos los tïtulos de los encabezados de las diferentes secciones de un texto mediante los
                    enczabezados. Tenemos seis elementos de encabezados
                        <h1>,<h2>,<h3>,<h4>,<h5>,<h6>
                

                
El titulo de este documento esta marcado con <h1>
                

                
                    

                
El parrafo es un elemento de bloque, siempre comienza en una linea y el navegador genera un
                    margen de 16 pixeles entre el contenido anterior y posterior al parrafo. A veces queremos saltar
                    una linea en un parrafo sin tener un parrafo nuevo. Podemos usar para ello la etiqueta <pre>
                

                
                

                
A veces queremos que una parte de un parrafo tenga una presentación diferente del resto
                    Usamos para ello la etiqueta <span> junto con el atributo class
                    para asosiar la clase de la hoja de estilos
                    que queremos dar a esa porción independiente de texto. Para que en este parrafo la etiqueta span
                    salga resaltada en rojo, hemos usado <span class="etiqueta">
                

                
Listas

                
Las Listas las usamos continuamente para enumerar una serie de elementos (personas, ingredientes,
                    provincias...) que guardan una relacción entre ellos 

                
                

                
Listas desordenadas

                
Las Listas no ordenadas se usan para marcar listas de articulos cuyo orden no es importante.

                
                

                
Cada lista desordenada comienza con un elemento <ul> que
                    determina todos los elementos de la lista. Cada elemento de la lista esta delimitado por una
                    etiqueta <li>

                
Ejemplo:

                
                    
                        <ul>
                            <li>Leche<li>
                            <li>Huevos<li>
                            <li>pan<li>
                            <li>hummus<li>
                        <ul>
                    
                

                
La presentación por defecto de la lista anterior sería

                
                

                
    
                    
  • leche
    • 
                    
  • huevos
    • 
                    
  • pan
    • 
                    
  • hummus
    •

      Listas de descripciones
      
                
       El propósito de estas listas es marcar un conjunto de elementos y sus descripciones asociadas,
                        como términos y definiciones, o preguntas y respuestas. Veamos un ejemplo de un conjunto de
                        términos y definiciones:
      
                        
                
      Las listas de descripción utilizan un contenedor diferente al de los otros tipos de listas — 
                        («description list» o lista de descripciones); además, cada término está envuelto en un elemento
                        <dt> («description term» o término a describir), y cada descripción está envuelta en un elemento
                        <dd> («description definition» o definición de descripción).Ejemplo:
      
                 
      		
         <dl>
        <dt>soliloquio</dt>
         <dd>En las obras dramáticas, corresponde a cuando un personaje se habla a sí mismo</dt>
        <dt> monólogo  </dt>
        <dd>Cuando un personaje habla de sus pensamientos en voz alta para compartirlo<dd>
        </dt>
		
                              
		
                        El resultado de la lista anterior es
               
      	       
  			  
      
                        
      soliloquio
      
                        
      En las obras dramáticas, corresponde a cuando un personaje se habla a sí mismo
                        
       monólogo  
      
                        
      Cuando un personaje habla de sus pensamientos en voz alta para compartirlo
      
                        
			
      
				
                  
                        
      Enfasis e Importancia
      
                        
      
                        
      Enfasis
      
                        
      Cuando queremos dar énfasis al lenguaje hablado, acentuamos ciertas palabras y así alteramos
                            sutilmente el significado de lo que decimos. De manera similar, en el lenguaje escrito
                            ponemos
                            palabras en cursiva para destacarlas. Por ejemplo, las dos siguientes frases tienen
                            diferente
                            significado:
      
                        
      
                        Me alegro de que no llegues tarde.
                        
      
                        Me alegro de que no llegues tarde.
                        
      
                        
      En HTML usamos el elemento <em>(«emphasis») para marcar
                            estos casos.
      
                        
      
                        
      El documento logra entonces transmitir una lectura más interesante y además así lo reconocen
                            los
                            lectores de pantalla, que lo expresan con un diferente tono de voz. El navegador, de manera
                            predeterminada, aplica el estilo de letra itálica, pero no debes utilizar esta etiqueta
                            solamente para
                            establecer el estilo de letra itálica. Para usar ese estilo, debes utilizar únicamente la
                            etiqueta del
                            elemento <span> y algo de CSS u otra etiqueta con el
                            elemento 
      

                        
      Importancia Fuerte
      
                        
      
                        
      Para enfatizar palabras importantes al hablar solemos acentuarlas, y al escribir lo hacemos
                            en
                            estilo negrita. Por ejemplo
      

                        
      Este líquido es altamente toxico
                        
      
                        
      Cuento contigo. ¡No llegues tarde!
      
                        
      
                        
      En HTML usamos el elemento <strong>(importancia fuerte) para marcar tales expresiones
                        
      

                        
      Negrita, cursiva y subrayado
      
                        
      
                        
       Si queremos poner un texto en cursiva porque tradicionalmente se hace asi, como por ejemplo
                        en
                        los nombres científicos de la especies, usaremos la etiqueta <i> puesto que en este caso
                        la
                        semántica es diferente. En este caso no queremos enfatizar.
      

                        
      Ejemplo: Lobo (Canis Lupus)

                        
      Lo mismo se aplica para el subrayado mediante la etiqueta 
                                <u> y para la negrita mediante la
                            etiqueta  <b>


                        
      Abrebiaturas
      
                        
      
                        
      Usaremos la etiqueta <abbr> para envolver una abreviatura
                            o acrónimo y proporcionar la
                            expansión del término mediante el atributo title.
      

                        
      
                        
      Ejemplo: Estamos cursando el primer curso del ciclo ASIR.
      
                        
      Informacion de contacto
      
                        
      

                        
      Usaremos la eligueta <aduress> para envolver la mlormación de contacto,
      

                        
      Superindices y subindices
      
                        
      

                        
      Usaremos la etiqueta <sup> para escribir superíndices. y la etiqueta <sub> para
                            marcar un
                            subíndice.
      

                        
      Ejemplo
      

                            
      X2
